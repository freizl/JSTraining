## grunt tasks
  - `bbb server` : preview in dev
  - `bbb debug` : merge js, merge css, etc.
  - `bbb server:debug` : preview base on build result of `bbb debug`.
  - `bbb release` : minify result of `bbb debug`.
  - `bbb server:release`
